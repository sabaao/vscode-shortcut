# vscode-shortcut

# 操作
## 上一頁
```
Fn + ↑
```

## 下一頁
```
Fn + ↓
```
## 切換到右邊的程式
```
option + command + →
```

## 切換到左邊的程式
```
option + command + ←
```
## 打開explorer
```
shift + command + e
```

## 全域搜尋
```
shift + command + f
```
## 打開method清單
```
shift + command + o
```

## 回上一個游標
```
command + u
```

## 搜尋檔案
```
command + p
```

## format code
```
shift + option + F
```
## into implement
```
Fn + F12
```

## 回上一個位置
```
control + -
```

## extract method
```
command + .
```

## rename
```
Fn + F2
```

## Quick fix
```
command + .
```

## Find next
```
command + shift + G
```

## Add selection to next Find match
```
command + D
```

## Go to File...
```
command + P
```

## Go to Symbol...
```
command + Shift + O
```

## Go to next/previous error or warning
```
F8 / Shift + F8
```

## Go back/forward
```
control + - / control + Shift + -
```
