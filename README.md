# vscode-shortcut

# 操作
## 上一頁
```
Fn + ↑
```

## 下一頁
```
Fn + ↓
```
## 切換到右邊的程式
```
option + command + →
```

## 切換到左邊的程式
```
option + command + ←
```
## 打開explorer
```
shift + command + e
```

## 全域搜尋
```
shift + command + f
```
## 打開method清單
```
shift + command + o
```

## 回上一個游標
```
command + u
```

## 搜尋檔案
```
command + p
```

## format code
```
shift + option + F
```
## into implement
```
Fn + F12
```
